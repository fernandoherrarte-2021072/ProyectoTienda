# Sistema de Ventas - Proyecto Tienda

Sistema web de ventas desarrollado como proyecto académico, orientado a la gestión de productos, clientes, ventas y usuarios.
El sistema implementa control de roles para diferenciar accesos entre administradores y vendedores.


## Tecnologías Utilizadas

- **Java 21**
- **Spring Boot 4.0.2**
- **Maven** (Gestor de dependencias)
- **MySQL** (Sistema Gestor de Base de Datos)
- **Thymeleaf** (Motor de plantillas)
- **HTML5 / CSS3**
- **Git & GitHub** (Control de versiones)

## Requisitos Previos

Antes de ejecutar el proyecto es necesario contar con:

- **JDK 21** o superior instalado
- **Maven** instalado y configurado
- **MySQL Server** en ejecución
- Un IDE compatible como **IntelliJ IDEA** o **Eclipse**
